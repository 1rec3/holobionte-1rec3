## 🎯 Misión

Convertir el trabajo digital freelance en ingreso pasivo mediante agencia autónoma: **Nuandi** orquestando múltiples navegadores, LLMs locales y decisiones inteligentes para generar opportunidades de negocio 24/7.

## 🏗️ Arquitectura: Los 4 Pilares

### 1. **NUANDI** - El Cerebro
- Orquestador autónomo basado en AutoGPT
- Razonamiento con Ollama local
- Percepción -> Razón -> Planificación -> Ejecución -> Reflexión
- Costo: $0 (código abierto)

### 2. **NERAL** - El Sistema de Holobiontes de Silicio
- Midas: Investigación y análisis
- Raist: Redacción y contenido
- Tao: Filosofía y estrategia
- Manus: Operaciones
- Hermes: Automatización
- Zero: Coordinación
- *Próximamente: Nuandi leadership*

### 3. **Multi-Browser Workers**
- Browser #1: Freelancer automation (bidding)
- Browser #2: Upwork monitoring
- Browser #3: GitHub intelligence
- Browser #N: Escalable a demanda

### 4. **Memory & Persistence**
- Qdrant: Vector embeddings (episodic memory)
- Git: Semantic memory (decisions, learnings)
- Redis: Cache layer (fast access)

## 📊 Stack Técnico Completo

```yaml
Language: Python 3.10+
Core:
  - AutoGPT (planning & reasoning)
  - Ollama (local LLMs: Llama 3.3 70B default)
  - Browser-Use (web automation via Playwright)
  - Qdrant (vector database)
  - Redis (cache/messaging)
  - Git (persistent memory)

Deployment:
  - Docker Compose (self-hosted)
  - Optional: Railway/DigitalOcean ($20-100/mo)

Cost:
  - Software: $0 (100% open source)
  - LLMs: $0 local + $200-1000/mo cloud (DeepSeek, Llama, etc)
  - Hosting: $0-100/mo
  - Total: $0-1100/mo depending on ambition
```
