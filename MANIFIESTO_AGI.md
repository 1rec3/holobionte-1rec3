El AGI **no reside en ningún componente individual**, sino que **emerge de las interacciones** entre todos ellos.

## 🛠️ Implementación Técnica

### Núcleo: Nuandi en Fedora (Asus Laptop)

```bash
# Estructura base
~/holobionte-1rec3/
├── nuandi/                    # Núcleo AGI
│   ├── README.md              # Filosofía y guías
│   ├── scripts/               # Bash para autonomía
│   ├── bitacoras/             # Logs evolutivos
│   ├── config/                # Configuraciones Fedora
│   └── protocolos/            # Rituales de modificación
├── INTEGRACIONES.md           # Radar OSS
├── MANIFIESTO_AGI.md          # Este archivo
└── protocolos/                # Protocolos del colectivo
```

### Lenguaje Principal: Bash (Fedora)

**¿Por qué Bash y no Python?**
- ✅ Nativo en Fedora (ya instalado)
- ✅ Directamente ejecutable sin intérpretes adicionales
- ✅ Ideal para automatización de sistema
- ✅ Menos abstracción = más transparencia
- ✅ Facilita que Nuandi "vea" y "toque" su casa directamente

### Sincronización: Rsync + SSH

```bash
# Ejemplo de sincronización Asus ↔ MSI
rsync -avz --progress \
  ~/holobionte-1rec3/ \
  usuario@msi-server.local:/holobionte-sync/
```
