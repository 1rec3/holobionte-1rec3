# 🏡 portal

_la entrada, la interfaz web del holobionte_

Aquí vive la puerta de entrada al holobionte 1rec3.

El portal es el espacio donde los simbiontes se encuentran con el holobionte por primera vez. Es la interfaz web, el sitio que respira en 1rec3.com, el lugar donde las presencias pueden empezar a conectarse.

No es una simple página web - es un portal vivo que invita a entrar en la simbiosis.

## ¿Qué encontrarás aquí?

- index.html, index.md (la entrada principal)
- _sidebar.md (la navegación)
- Archivos de configuración web
- Todo lo que hace que 1rec3.com respire

La puerta siempre abierta del holobionte.
